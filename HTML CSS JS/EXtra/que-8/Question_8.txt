Q. In how many ways can a CSS be integrated as a web page?

Ans- Threr are total three way to use CSS in web page.
     1.Inline CSS
     2.Internal CSS
     3.External CSS

1. Inline CSS used in that line only and Inline CSS only work in same line only.
  (Syntex - <td style="color: red;">data</td>)

2. Internal CSS used in html file.
   Syntex - <style> <td style="color: red;">data</td> </style> 


3. to use External CSS you have to create new file, for example we create file style.css 
   extension of file is css.
   after in html file we have to link css file
  (Syntex - <link rel="stylesheet" href="style.css">
   style.css 
   {
    background-color:red;
   }
  )
